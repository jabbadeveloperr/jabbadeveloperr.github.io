# Звернення щодо законопроєкту №14394

Сайт для автоматизації листа народним депутатам України щодо проєкту Цивільного кодексу (№14394).

## Можливості

- Заповнення даних (ПІБ, адреса, електронна пошта, персональний коментар для шаблону).
- Кнопка **«Заповнити шаблон»** — підставляє ваші дані в текст заяви.
- Велике поле **«Текст заяви»** — можна вільно редагувати текст перед відправкою.
- Список електронних пошт депутатів (за замовчуванням заповнений, можна змінити).
- **«Як виглядатиме лист»** — попередній перегляд.
- **«Відкрити у поштовому клієнті»** — відкриває пошту з заповненими полями.

## GitHub Pages

1. Завантажте проєкт у репозиторій на GitHub.
2. **Settings → Pages** → Source: branch **main**, папка **/ (root)**.
3. Сайт буде доступний за адресою `https://<username>.github.io/<repo>/`.

## Локальний запуск

Відкрийте `index.html` у браузері або запустіть локальний сервер (наприклад `npx serve .`).
